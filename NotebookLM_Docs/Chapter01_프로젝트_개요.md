# Chapter 01: SmartHandWash 프로젝트 개요

## 무엇을 만드나요?

**SmartHandWash**는 손 세정 스테이션을 컴퓨터 화면에서 똑같이 재현한 **디지털 트윈(Digital Twin)** 시뮬레이터입니다.

공항이나 병원에서 봤던 자동 손 세정기 기억나시나요? 센서 앞에 손을 대면 비누가 나오고, 물이 나오고, 에어로 말려주는 그 장비요. 이 프로젝트는 그 장비를 가상으로 만든 거예요.

---

## 디지털 트윈이란?

> 💡 **핵심 개념**: 디지털 트윈은 "현실의 쌍둥이"예요!

```
┌──────────────────┐              ┌──────────────────┐
│   실제 장비       │              │   디지털 트윈     │
│   (현실 세계)     │    ◀───▶    │   (가상 세계)     │
│                  │    실시간    │                  │
│  🧴 손 세정 스테이션│    동기화    │  🖥️ Unity 시뮬레이터│
└──────────────────┘              └──────────────────┘
```

실제 장비에서 비누가 50% 남았다면, 컴퓨터 화면에서도 50%로 표시됩니다.
화면에서 버튼을 누르면, 실제 장비에서도 작동할 수 있어요.

---

## 왜 디지털 트윈을 만드나요?

### 1. 원격 모니터링
공장에 직접 가지 않아도 장비 상태를 확인할 수 있어요.
서울 사무실에서 부산 공장의 손 세정기 상태를 볼 수 있죠.

### 2. 시뮬레이션 테스트
실제 장비 없이도 소프트웨어를 테스트할 수 있어요.
"비누가 0%일 때 어떻게 되지?" 같은 상황을 가상으로 실험합니다.

### 3. 교육 및 훈련
신입 직원에게 실제 장비 조작법을 가르치기 전에,
가상 환경에서 먼저 연습할 수 있어요.

## 실제 사용 시나리오

### 시나리오 1: 비누 사용
1. 사용자가 **비누 버튼**을 클릭
2. 비누가 **3초간** 분사됨 (파티클 효과)
3. 비누 잔량이 **5%** 감소
4. 잔량이 **20% 이하**면 → 주의(🟡) 표시
5. 잔량이 **0%**면 → 오류(🔴) 표시, 비누 버튼 비활성화

### 시나리오 2: 인터락 작동
1. 비누가 분사되는 **도중에**
2. 물 버튼을 누르면?
3. → **아무 일도 안 일어남!** (인터락 시스템)
4. 비누 분사가 끝난 후에야 물 버튼이 작동함

## 자주 묻는 질문

**Q: 왜 Unity로 만들었나요?**

A: Unity는 3D 시각화에 강력한 게임 엔진이에요.
   파티클 효과, 실시간 렌더링, 크로스 플랫폼 지원이 뛰어나서
   산업용 시뮬레이터 개발에도 많이 사용됩니다.

**Q: 실제 PLC 없이도 작동하나요?**

A: 네! **Mock 모드**가 있어서 가상 PLC로 테스트할 수 있어요.
   실제 장비 연결은 나중에 설정으로 전환할 수 있습니다.
