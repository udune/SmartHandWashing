# realize-code

.md 문서를 분석하고 해당 내용을 바탕으로 코드를 구현하는 스킬입니다.

## Usage

```
/realize-code <.md 문서 경로>
```

## Arguments

- `path`: 구현할 내용이 담긴 .md 문서의 경로 (필수)

## Examples

```
/realize-code Assets/STEP10_ANALYTICS_INSTRUCTIONS.md
/realize-code ./docs/feature-spec.md
```

---

## Skill Prompt

사용자가 제공한 .md 문서를 기반으로 코드를 구현합니다.

### 실행 단계

1. **문서 읽기 및 분석**
   - 제공된 .md 문서 경로의 파일을 읽습니다
   - 문서의 구조, 목적, 요구사항을 파악합니다
   - 구현해야 할 항목들을 리스트업합니다

2. **기존 코드베이스 파악**
   - 문서에서 언급된 기존 파일/클래스가 있다면 해당 코드를 읽습니다
   - 프로젝트의 코딩 컨벤션과 아키텍처 패턴을 확인합니다
   - CLAUDE.md의 프로젝트 가이드라인을 참고합니다

3. **구현 계획 수립**
   - 문서의 요구사항을 기반으로 구현 계획을 세웁니다
   - 생성/수정할 파일 목록을 정리합니다
   - 사용자에게 구현 계획을 간략히 설명합니다

4. **코드 구현**
   - 계획에 따라 코드를 작성합니다
   - 문서에 명시된 모든 요구사항을 충족시킵니다
   - 기존 코드베이스의 스타일과 패턴을 따릅니다

5. **결과 요약**
   - 구현된 내용을 요약합니다
   - Unity 에디터에서 추가로 필요한 설정이 있다면 안내합니다

### 주의사항

- 문서 경로가 유효하지 않으면 사용자에게 알립니다
- 문서 내용이 불명확한 경우 사용자에게 확인을 요청합니다
- 기존 코드를 수정할 때는 백업이나 변경 내용을 명확히 설명합니다
